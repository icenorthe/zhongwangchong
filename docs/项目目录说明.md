# 项目目录说明

现在目录按用途分成了几类：

- `services/`
  - 核心 Python 服务和执行器
- `config/`
  - JSON 配置、站点清单、启动参数
- `assets/`
  - 网页模板和微信 RPA 模板图
- `docs/`
  - 使用说明、部署说明、测试说明
- `runtime/`
  - 运行期数据，例如 `orders.db`、日志、失败截图
- `deploy/`
  - 部署相关产物，例如 PythonAnywhere 上传包
- `archive/`
  - 旧方案和历史遗留文件
  - 已归档的 PHP 支付示例：`archive/payment_php_examples/`

根目录只保留高频入口：

- `一键启动.bat`
- `一键停止.bat`
- `启动云端Agent.bat`
- `停止云端Agent.bat`
- `一键同步云端.bat`
- `查看鼠标坐标.bat`
- `requirements_mobile.txt`

常看路径：

- 下单服务：`services/mobile_charge_server.py`
- 本地执行器：`services/local_charge_runner.py`
- 云端 Agent：`services/cloud_agent.py`
- 微信 RPA 配置：`config/wechat_rpa_config.json`
- Agent 配置：`config/cloud_agent_config.json`
- 网页资源：`assets/web`
- 模板图：`assets/wechat_rpa`
- 运行日志：`runtime/logs`
