# 一键启动说明

## 文件

- `一键启动.bat`：启动本地桥接 + 下单服务 + 隧道
- `一键停止.bat`：停止上述服务
- `config/launcher_config.bat`：启动参数配置

## 使用方法

1. 双击 `一键启动.bat`
2. 等待显示 `启动检查完成`
3. 打开：
   - 下单页：`http://你的IP:8000`
   - 后台页：`http://你的IP:8000/admin`

## 首次检查

- 确保能执行：`python --version`
- 若使用 cloudflared 隧道，确保能执行：`cloudflared --version`
- 未安装 cloudflared 也可以先用本地/局域网访问，脚本会自动跳过隧道

## 常见问题

### 1) 端口被占用

双击 `一键停止.bat` 后，再双击 `一键启动.bat`。

### 2) 隧道未启动

查看 `runtime/logs/tunnel.log`。  
如果你不是 cloudflared，把 `config/launcher_config.bat` 中的 `TUNNEL_COMMAND` 改为你的命令。

### 3) 校园网 IP 经常变化

不要发 IP 给用户，发隧道域名。  
隧道域名请在 `runtime/logs/tunnel.log` 查看（cloudflared 启动后会打印）。
